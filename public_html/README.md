# PHP-версия сайта для обычного хостинга

Эту папку можно загружать в корень сайта на обычном PHP-хостинге, например в `public_html` на Beget.

## Что внутри

- `index.php` — единая точка входа и роутинг страниц.
- `app/` — PHP-логика сайта и админки.
- `data/site.json` — контент сайта, страницы, настройки и отзывы.
- `data/leads.json` — заявки.
- `uploads/` — загруженные картинки.
- `assets/` — стили и статические изображения.

## Перед загрузкой

1. Открой `app/config.php`.
2. Поменяй `ADMIN_PASSWORD` на сложный пароль.
3. Загрузи содержимое этой папки в корень сайта.
4. Убедись, что PHP может писать в папки:
   - `data/`
   - `uploads/`

## Админка

Админка доступна по адресу:

```text
/admin
```

Сложные блоки страниц редактируются как JSON. Это проще и надежнее для первой PHP-версии под обычный хостинг.
